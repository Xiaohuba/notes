# 计算机的一切
- 现代计算机的组成：1.运算器  2.控制器 3.存储器 4.输出设备 5.输入设备 
- cpu = 运算器+控制器 
- 运算器：实现逻辑运算和数字运算   ALU加法运算器 
- 控制器：控制协调计算机其他硬件的  寄存器属于控制器  IR：指令寄存器  PC：程序计数器 
- 存储器：内存和外存
- 输出设备：显示器
- 输入：键盘，鼠标 
- 控制器先从存储器找需要的指令，如果有多条指令，将取出的指令暂时放在 IR，执行一个动作，PC+ 1
- 把指令都取出来了之后，将所有指令送去译码器翻译，译码器产生电流脉冲，刺激其他硬件

# 原反补
- 正数加上正数，或者负数加上负数，就可能产生溢出

- 原码:所有原本的二进制数 反码:正数的反码是他本身，负数的反码是除了符号位以外，每一位取反 补码:正数的补码使他本身，负数的补码是反码加1

- bcd码是用四位二进制数表示一位十进制数的编码方式，每两个相邻的数字，只有一位不同

- e.g.补码：1111011 反码：1111010 原码：1000101 = -5
